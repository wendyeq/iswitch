# Code Switch 性能基准测试

> 记录 Go 版本和 Rust 版本的性能对比数据

## 测试方法

### 代理延迟测试

```bash
# 使用 hey 进行压力测试
hey -n 1000 -c 10 http://127.0.0.1:18099/v1/messages

# 或使用 wrk
wrk -t4 -c10 -d30s http://127.0.0.1:18099/v1/messages
```

### 内存占用测量

- 工具: Activity Monitor (macOS) / Task Manager (Windows)
- 状态: 空闲 / 满载 (100 并发请求)

### 安装包大小

- 命令: `ls -lh dist/*.dmg` (macOS)
