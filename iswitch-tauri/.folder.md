# iSwitch 应用根目录

## 功能定位

[POS]: iSwitch 桌面应用的顶级项目容器，采用 Rust (Tauri) + Vue 3 混合架构，提供 AI 提供商管理和本地代理服务

## 目录结构

| 目录/文件         | [POS] 定位                                      |
| :---------------- | :---------------------------------------------- |
| `src-tauri/`      | Rust 后端 - Tauri v2 应用、代理服务器、业务逻辑 |
| `src/`            | Vue 3 前端 - UI 界面、状态管理、用户交互        |
| `package.json`    | Node.js 依赖管理、npm scripts 配置              |
| `tauri.conf.json` | Tauri 应用配置 (窗口、权限、打包)               |
| `vite.config.ts`  | Vite 构建工具配置                               |
| `index.html`      | 前端入口 HTML                                   |
| `README.md`       | 项目说明文档                                    |

## 架构概览

```
iswitch-tauri/
├── src-tauri/        # Rust 后端 (Tauri v2)
│   └── src/
│       ├── commands/ # Tauri Commands - 前后端桥接
│       ├── services/ # 业务逻辑层
│       ├── proxy/    # HTTP 代理服务 (axum)
│       ├── models/   # 数据模型
│       ├── db/       # SQLite 数据库
│       └── utils/    # 工具函数
│
└── src/              # Vue 3 前端
    ├── components/   # Vue 组件
    ├── services/     # 前端 API 服务层
    ├── router/       # Vue Router 路由
    ├── utils/        # 前端工具函数
    ├── types/        # TypeScript 类型定义
    ├── data/         # 静态数据配置
    └── icons/        # 图标映射
```

## 技术栈

- **后端**: Rust + Tauri v2 + axum + SQLite
- **前端**: Vue 3 + TypeScript + TailwindCSS + Vite
- **通信**: Tauri IPC (invoke/ events)
