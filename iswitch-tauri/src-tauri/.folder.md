# Tauri 后端目录

## 功能定位

[POS]: Rust 后端项目容器，包含源码、构建配置、资源文件和 Tauri 应用配置

## 目录结构

| 目录/文件         | [POS] 定位                              |
| :---------------- | :-------------------------------------- |
| `src/`            | Rust 源码主目录 (详见 `src/.folder.md`) |
| `Cargo.toml`      | Rust 依赖配置、包元数据                 |
| `build.rs`        | 构建脚本 - 编译时执行                   |
| `tauri.conf.json` | Tauri 应用主配置 (窗口、权限、打包)     |
| `capabilities/`   | Tauri 权限定义 (ACL)                    |
| `resources/`      | 应用资源文件 (图标、静态资源)           |
| `icons/`          | 应用图标 (多平台、多尺寸)               |
| `gen/`            | 代码生成目录 (Tauri 自动生成)           |
| `tests/`          | Rust 集成测试                           |

## 构建流程

```
tauri.conf.json → build.rs → src/ → 编译、链接、打包
```

## 配置说明

- **tauri.conf.json**:
  - 窗口大小、标题、开发者工具
  - 安全权限 (filesystem、shell、http)
  - 打包配置 (updater、system tray)
- **Cargo.toml**:
  - Rust 依赖 (tokio, axum, sqlite, tauri-plugin-\*)
  - 工作空间成员配置

## 规范遵循

- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- 构建目标: macOS、Windows、Linux
- 产物路径: `src-tauri/target/release/bundle/`
