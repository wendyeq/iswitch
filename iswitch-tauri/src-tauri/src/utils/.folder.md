# Utils 模块

## 功能定位

[POS]: 工具函数集合，提供路径处理、通配符匹配、费用计算等通用功能

## 文件清单

| 文件          | [POS] 定位                   | 迁移自                             |
| :------------ | :--------------------------- | :--------------------------------- |
| `mod.rs`      | 模块导出                     | -                                  |
| `paths.rs`    | 配置文件路径解析             | 各 Service 中的路径逻辑            |
| `wildcard.rs` | 通配符模型匹配               | `providerservice.go#matchWildcard` |
| `pricing.rs`  | Model Pricing 加载与费用计算 | `logservice.go#calculateCost`      |
| `security.rs` | 敏感信息脱敏 (mask_api_key)  | -                                  |

## 依赖关系

- [INPUT]: model-pricing.json 定价文件
- [OUTPUT]: ../services/, ../proxy/

## 工具函数示例

```rust
// paths.rs
pub fn iswitch_dir() -> PathBuf;          // ~/.iswitch/ 应用配置目录
pub fn claude_settings_path() -> PathBuf; // ~/.claude/settings.json
pub fn codex_config_path() -> PathBuf;    // ~/.codex/config.toml
pub fn old_code_switch_dir() -> PathBuf;  // ~/.code-switch/ 用于导入

// wildcard.rs
pub fn match_wildcard(pattern: &str, model: &str) -> bool;

// pricing.rs
pub fn calculate_cost(model: &str, usage: &TokenUsage) -> f64;

// security.rs
pub fn mask_api_key(key: &str) -> String;
```
