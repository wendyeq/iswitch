# Rust 后端根目录

## 功能定位

[POS]: Code Switch 应用的 Rust 后端入口，基于 Tauri v2 构建，负责桌面集成、代理服务和业务逻辑

## 目录结构

| 目录/文件   | [POS] 定位                                      |
| :---------- | :---------------------------------------------- |
| `main.rs`   | Tauri 桌面应用入口，窗口和托盘初始化            |
| `lib.rs`    | 库入口，注册所有 Commands、插件、菜单及窗口特效 |
| `error.rs`  | 统一错误类型定义 (AppError)                     |
| `commands/` | Tauri Commands 层 - 前后端桥接                  |
| `services/` | 业务逻辑服务层 - 核心功能实现                   |
| `models/`   | 数据结构定义 - 序列化/反序列化模型              |
| `proxy/`    | HTTP 代理模块 - axum 服务器与请求转发           |
| `db/`       | 数据库层 - SQLite 操作封装                      |
| `utils/`    | 工具函数 - 路径、通配符匹配、定价计算等         |

## 依赖关系

```
main.rs
    └── lib.rs
          ├── commands/ ←──────┐
          │       ↓            │ 调用
          ├── services/ ───────┘
          │       ↓
          ├── models/  ←── db/
          │       ↓
          ├── proxy/
          │       ↓
          └── utils/
```

## 迁移自

- `code-switch/main.go` → `main.rs`, `lib.rs`
- `code-switch/services/` → `services/`

## 规范遵循

- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- 日志使用 `tracing` crate
- 错误处理使用 `thiserror` + `AppError`
