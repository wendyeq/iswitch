# Models 模块

## 功能定位

[POS]: 数据结构定义，包含所有序列化/反序列化模型，确保与 Go 版本 JSON 格式兼容

## 设计原则

1. **JSON 兼容**: 使用 `#[serde(rename = "...")]` 确保字段名与 Go 版本一致
2. **类型安全**: 优先使用枚举类型而非字符串
3. **Default 实现**: 为配置类型实现 `Default` trait

## 文件清单

| 文件          | [POS] 定位             | 对应 Go 结构体               |
| :------------ | :--------------------- | :--------------------------- |
| `mod.rs`      | 模块导出               | -                            |
| `provider.rs` | Provider, ProviderKind | `Provider` struct            |
| `mcp.rs`      | MCPServer, MCPType     | `MCPServer` struct           |
| `skill.rs`    | Skill, SkillRepo       | `Skill`, `SkillRepo` structs |
| `log.rs`      | RequestLog, LogStats   | `RequestLog` struct          |
| `settings.rs` | AppSettings            | `AppSettings` struct         |

## 依赖关系

- [INPUT]: Go 版本的 struct 定义
- [OUTPUT]: ../services/, ../commands/, ../db/
