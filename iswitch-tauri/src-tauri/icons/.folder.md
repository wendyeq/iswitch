# 应用程序图标目录

## 功能定位

[POS]: 存储 iSwitch 应用程序在不同操作系统（macOS, Windows, iOS, Android）下的图标资源

## 资源列表

- `icon.png`: 原始高清图标 (1024x1024)
- `icon.ico`: Windows 图标文件
- `icon.icns`: macOS 图标文件
- `32x32.png`, `128x128.png` 等: 不同分辨率的 PNG 图标，供 Tauri 绑定包使用
- `ios/`, `android/`: 移动端平台特定图标

## 生成工具

- 使用 `npm run tauri icon` 命令从 `icon.png` 自动生成所有尺寸。

## 规范遵循
