# Vue Router 路由目录

## 功能定位
[POS]: 前端路由配置中心，定义应用页面导航结构

## 成员清单

| 文件 | [POS] 定位 |
| :--- | :--- |
| `index.ts` | 路由配置 - Vue Router 实例、路由表定义 |

## 路由结构

**主要路由**:
- `/` - 主页 (Provider 管理)
- `/logs` - 请求日志页面
- `/settings` - 设置页面
- `/mcp` - MCP 管理页面
- `/skill` - Skill 管理页面

## 路由配置
```typescript
const routes = [
  { path: '/', component: Main },
  { path: '/logs', component: Logs },
  { path: '/settings', component: Setting },
  // ...
]
```

## 导航模式
- 使用 Hash Router (Tauri 推荐)
- 支持浏览器后退/前进
- 路由守卫 (如需要)

## 依赖关系
```
App.vue → Router → Page Components
```

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0 - 分形自洽

- 组件命名使用 PascalCase
