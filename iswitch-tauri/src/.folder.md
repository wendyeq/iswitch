# Vue 3 前端源码目录

## 功能定位
[POS]: iSwitch 应用前端代码主目录，基于 Vue 3 + TypeScript + TailwindCSS，提供用户界面和交互逻辑

## 目录结构

| 目录/文件 | [POS] 定位 |
| :--- | :--- |
| `App.vue` | Vue 应用根组件 |
| `main.ts` | 前端入口 - Vue 实例初始化、插件挂载 |
| `components/` | Vue 组件库 (通用、页面级) |
| `services/` | 前端 API 服务层 - Tauri invoke 封装 |
| `router/` | Vue Router 路由配置 |
| `utils/` | 前端工具函数 (i18n、toast、theme、hotkey) |
| `types/` | TypeScript 类型定义 |
| `data/` | 静态数据配置 (卡片配置、热力图数据) |
| `icons/` | 图标映射 (LobeHub 图标集成) |
| `locales/` | 国际化语言包 (i18n) |
| `style.css` | 全局样式、TailwindCSS 配置 |

## 组件架构
```
App.vue
├── Main/        # 主页面
├── Setting/     # 设置页面
├── Logs/        # 日志页面
├── Mcp/         # MCP 管理页面
├── Skill/       # Skill 管理页面
└── General/     # 通用组件
```

## 数据流
```
用户操作 → Component → Service (Tauri invoke) → Rust Commands → Service Layer → Response
```

## 依赖关系
- **UI 框架**: Vue 3 (Composition API)
- **路由**: Vue Router 4
- **样式**: TailwindCSS
- **国际化**: Vue I18n
- **图标**: LobeHub Icons + SVG
- **状态**: Pinia (如需要)

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- TypeScript 严格模式
- 组件命名: PascalCase
- 样式优先: TailwindCSS 工具类
- 所有后端调用通过 `services/` 层
