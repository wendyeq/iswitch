# 前端 API 服务层目录

## 功能定位
[POS]: Tauri Commands 调用封装层，为前端组件提供类型安全的 API 接口

## 成员清单

| 文件 | [POS] 定位 |
| :--- | :--- |
| `tauri.ts` | Tauri 基础封装 - invoke、event、channel 通用方法 |
| `appSettings.ts` | 应用设置服务 - 端口、主题、语言等配置 |
| `claudeSettings.ts` | Claude 设置服务 - Claude 相关配置管理 |
| `logs.ts` | 日志服务 - 请求日志查询、筛选、导出 |
| `mcp.ts` | MCP 服务 - MCP 服务器管理、配置 |
| `skill.ts` | Skill 服务 - Skill 管理功能 |
| `version.ts` | 版本服务 - 应用版本检查、更新 |
| `configImport.ts` | 配置导入服务 - 从 Code Switch 导入配置 |

## 服务模式

**调用流程**:
```typescript
// 前端组件
import { getProviders } from '@/services/tauri'

const providers = await getProviders() // invoke('get_providers')
```

**统一错误处理**:
- 所有服务调用捕获 Tauri 错误
- 返回类型安全的响应
- 提供 Toast 提示

## 依赖关系
```
Components/ → services/ → Tauri Commands (Rust)
```

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- 每个服务对应一组相关 Commands
- 函数命名与 Rust Commands 保持一致 (camelCase)
- 完整 TypeScript 类型定义
