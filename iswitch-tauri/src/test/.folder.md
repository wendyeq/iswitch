# Test Configuration

> **POS**: iswitch-tauri/src/test/

本目录包含 Vitest 测试环境的全局配置和辅助工具。

## 核心文件

| 文件 | 描述 |
|:-----|:-----|
| `setup.ts` | Vitest 全局设置，负责 Mock Tauri API、浏览器环境 (localStorage, matchMedia) 等 |

## 测试策略

- **Mocking**: 使用 `vi.mock` 模拟 `@tauri-apps/api/core` 以避免真实 IPC 调用。
- **Environment**: 使用 `jsdom` 模拟浏览器 DOM 环境。
