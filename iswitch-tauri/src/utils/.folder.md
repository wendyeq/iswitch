# 前端工具函数目录

## 功能定位

[POS]: 前端通用工具函数集合，提供国际化、主题管理、快捷键、通知等跨组件功能

## 成员清单

| 文件                  | [POS] 定位                                     |
| :-------------------- | :--------------------------------------------- |
| `i18n.ts`             | 国际化工具 - Vue I18n 配置、语言切换逻辑       |
| `toast.ts`            | 通知工具 - 统一的消息提示接口                  |
| `ThemeManager.ts`     | 主题管理器 - 系统/深色/浅色主题切换、持久化    |
| `hotkeyUtils.ts`      | 快捷键工具 - 全局快捷键注册、监听、管理        |
| `faviconFetcher.ts`   | 图标获取 - Unavatar Favicon 服务封装与智能降级 |
| `providerDetector.ts` | 供应商检测 - URL 解析与服务商识别              |

## 功能分类

**国际化**:

- 语言包加载
- 动态切换语言
- 格式化日期、数字

**主题**:

- 自动跟随系统主题
- 手动切换深色/浅色
- 主题持久化存储

**交互**:

- Toast 消息提示 (成功/错误/警告)
- 全局快捷键绑定
- 防冲突快捷键管理

**数据/资源**:

- Favicon 智能获取
- API 服务商自动识别

## 依赖关系

```
Components/ → utils/ (调用工具函数)
```

## 规范遵循

- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- 纯函数优先，无副作用
- 统一错误处理
