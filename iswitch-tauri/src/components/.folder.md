# Components 目录

## 功能定位

[POS]: Vue 组件库根目录，按功能模块组织页面级组件和通用 UI 组件。

## 目录结构

| 目录        | [POS] 描述                                       |
| :---------- | :----------------------------------------------- |
| `common/`   | 通用基础组件 (Button、Input、Modal、Textarea 等) |
| `HUD/`      | Mini HUD 悬浮窗组件                              |
| `Logs/`     | 日志查看器组件                                   |
| `Main/`     | 主页 (Hero) 相关组件                             |
| `Mcp/`      | Model Context Protocol 管理组件                  |
| `Modals/`   | 全局模态框组件 (如端口冲突提示)                  |
| `Settings/` | 设置页面组件（现存唯一设置实现）                 |
| `Skill/`    | Skill 扩展管理组件                               |

## 依赖关系

- [INPUT]: `../App.vue` '引入页面级组件'
- [OUTPUT]: 各子目录组件实现
