# HUD 组件

## 功能定位

[POS]: Mini HUD 浮动窗口组件，显示实时 AI 请求统计信息

## 核心功能 (Jony Ive Style)

1. **极简三层布局**: Hero (Speed 42px) → Secondary (Tokens 22px) → Tertiary (Model 11px)
2. **无标签无边框**: 数字直接漂浮在磨砂玻璃上，发丝分割线区隔层级
3. **容器呼吸光晕**: Streaming 时整个玻璃容器呼吸发光
4. **圆形图钉锁定**: 22px 圆形按钮，锁定时实心 + 发光效果

## 文件清单

| 文件        | [POS] 定位                         |
| :---------- | :--------------------------------- |
| `Index.vue` | HUD 主组件，处理事件监听和 UI 渲染 |

## 依赖关系

- [INPUT]: Tauri 事件 `hud-update` - 接收后端推送的 HUD 更新事件
- [INPUT]: `@tauri-apps/api/core` - 调用 HUD 窗口控制命令
- [OUTPUT]: 可视化的实时 AI 请求统计

## 事件格式

```typescript
interface HudEvent {
  provider: string;
  model: string;
  delta_tokens: number;
  total_tokens: number;

  speed: number;
  status: 'starting' | 'streaming' | 'completed' | 'error';
}
```
