
# Main 组件目录

## 功能定位
[POS]: 主界面核心组件，包含英雄区(Hero)、热力图(Heatmap)、控制区(Controls)和主要的自动化列表(Automation List)。

## 目录结构

| 目录/文件 | [POS] 定位 |
| :--- | :--- |
| `Index.vue` | 主页入口组件 (Main Page) |
| `LevitatingCapsule.vue` | 悬浮胶囊单项组件 (Item) |
| `LevitatingProviderList.vue` | 悬浮胶囊列表容器 (List) |
| `CapsuleNavigation.vue` | 底部悬浮导航条 (Navigation) |
| `VendorReceptacle.vue` | 智能供应商接收槽 (Vendor Form Replacement) |

## 依赖关系
- **Parent**: `App.vue`
- **Data**: `cards.ts` (Provider Data), `usageHeatmap.ts` (Heatmap Data)
- **Services**: `logs.ts`, `claudeSettings.ts`

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0
- 组件通信：Props down, Events up
- 样式：TailwindCSS + CSS Variables (Theme)
