# Logs 日志组件

## 功能定位

[POS]: 记录请求日志的仪表盘页面，提供汇总统计、过滤表单与表格明细。

## 文件清单

| 文件            | [POS] 描述                                          |
| :-------------- | :-------------------------------------------------- |
| `Index.vue`     | 页面根组件，渲染日志摘要、Chart.js 折线图与请求表格 |
| `Index.test.ts` | Logs 页面单元测试（布局与渲染场景）                 |

## 依赖关系

- [INPUT]: `../../services/logs` 获取日志列表、统计与供应商
- [INPUT]: `../common/BaseButton.vue`、`vue-chartjs` 渲染交互与图表
- [OUTPUT]: 提供 `/logs` 路由 UI，支撑诊断工作流

## 设计提示

- [PROTOCOL]: FractalFlow v1.0
- 共用 `main-shell` + `contrib-page` 布局，颜色必须引用 Prism 主题 Token
- Chart.js 配色通过主题 Token 与 `useThemeTokens` 助手同步
