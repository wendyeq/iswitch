# Modals 目录

## 功能定位

[POS]: 存放全局弹窗组件（如端口冲突、告警提示等），与页面组件解耦，便于统一样式与逻辑复用。

## 成员清单

| 文件               | [POS] 描述                                              |
| :----------------- | :------------------------------------------------------ |
| `PortConflict.vue` | 监听 `proxy-error` 事件并向用户展示端口被占用的处理流程 |

## 依赖关系

- [INPUT]: `src-tauri/src/lib.rs` 发射的事件、`@tauri-apps/api/event`
- [OUTPUT]: `App.vue` 根组件引入后渲染模态框

## 规范

- 遵循 FractalFlow Header（含 `[INPUT]` / `[OUTPUT]` / `[POS]` / `[PROTOCOL]`）
- 所有注释使用简体中文
