# Settings Component Architecture

## Directories
- `.`

## Files
- `CrystalControl.vue`: Main settings panel container with crystal glass effect
- `PhysicSwitch.vue`: Skeuomorphic toggle switch with physics
- `MemoryCrystal.vue`: Status indicator component for sync/import
- `Section.vue`: Reusable settings section component
- `SyncCrystal.vue`: Sync status visualization component

## Architecture
- **State Management**: Local reactive state initialized from `fetchAppSettings`.
- **Persistence**: `saveAppSettings` called on change.
- **Design System**: "Crystal Control" aesthetic - Glassmorphism, transparency, light/dark mode.
