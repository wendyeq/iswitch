# 静态数据配置目录

## 功能定位
[POS]: 前端静态数据中心，存储 UI 配置、常量数据和演示数据

## 成员清单

| 文件 | [POS] 定位 |
| :--- | :--- |
| `cards.ts` | 卡片配置 - 主页卡片布局、图标、路由定义 |
| `usageHeatmap.ts` | 热力图数据 - 使用量统计热力图的演示数据 |

## 数据说明

**卡片配置**:
- 主页功能卡片列表
- 每个卡片包含: 标题、描述、图标、路由
- 支持动态显示/隐藏

**热力图数据**:
- Provider 使用量热力图数据
- 用于 UI 演示和测试
- 生产环境应从后端 API 获取

## 依赖关系
```
Components/ → data/ (读取静态配置)
```

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0 - 分形自洽
- 配置数据使用 TypeScript 常量
- 支持类型推断
- 易于维护和扩展
