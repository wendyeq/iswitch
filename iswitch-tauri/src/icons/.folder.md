# 图标映射目录

## 功能定位
[POS]: LobeHub Icons 集成层，提供 AI 服务商图标映射和回退方案

## 成员清单

| 文件 | [POS] 定位 |
| :--- | :--- |
| `lobeIconMap.ts` | 主图标映射表 - 提供商名称到 LobeHub Icon 的映射 |
| `fallbackLobeIcons.ts` | 回退图标配置 - 缺失图标时的默认图标列表 |

## 功能说明

**图标映射**:
- 根据 Provider 名称匹配 LobeHub Icon
- 支持模糊匹配 (例如: "OpenAI" → "openai")
- 大小写不敏感

**回退机制**:
- 当 LobeHub Icons 无匹配时使用默认图标
- 支持自定义回退图标集
- 保证 UI 不会因缺失图标而破坏

## 使用示例
```typescript
import { getIcon } from '@/icons/lobeIconMap'

const icon = getIcon('Anthropic') // 返回对应 SVG 组件
```

## 依赖关系
```
components/ → icons/ (获取图标组件)
```

## 规范遵循
- [PROTOCOL]: FractalFlow v1.0 - 分形自洽

- 优先使用官方图标，回退使用通用图标
