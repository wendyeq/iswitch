# Docs 文档目录

## 功能定位

[POS]: 项目开发者文档目录，存放架构设计、API 文档和开发指南

## 文件清单

| 文件                       | [POS] 定位                                                     |
| :------------------------- | :------------------------------------------------------------- |
| `ARCHITECTURE.md`          | 项目整体架构文档 - 系统概览、目录结构、数据流、核心模块说明    |
| `FRONTEND_ARCHITECTURE.md` | 前端架构文档 - Vue 3 组件结构、服务层、数据流、待优化项        |
| `versioning.md`            | 版本管理工作流文档 - 版本号规范、发布流程、Commit Message 规范 |
| `QUALITY_ASSURANCE_SOP.md` | 质量保障 SOP - 开发工作流、测试标准、Code Review 规范          |

## 维护规范

- 架构变更时需同步更新文档
- 使用 ASCII 艺术图表保持可读性和版本控制友好
- 所有描述使用简体中文
