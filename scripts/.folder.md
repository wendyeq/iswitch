# 脚本目录

## 简介

本目录包含项目构建、版本管理和自动化相关的脚本文件。

## 文件说明

| 文件 | 用途 |
|------|------|
| `bump-version.sh` | 版本号同步脚本 (macOS/Linux) |
| `bump-version.ps1` | 版本号同步脚本 (Windows PowerShell) |

## 使用方式

推荐通过 `Makefile` 调用这些脚本，例如：

```bash
make bump-patch   # 递增补丁版本 0.1.0 → 0.1.1
make bump-minor   # 递增次版本   0.1.0 → 0.2.0
make bump-major   # 递增主版本   0.1.0 → 1.0.0
make version      # 显示当前版本
```

## 依赖

- `jq`: 用于 JSON 文件处理 (macOS 可通过 `brew install jq` 安装)
- `sed`: 用于 TOML 文件处理 (macOS 自带)
